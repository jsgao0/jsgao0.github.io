因為常常看到許多大大用Jekyll或Hexo在Github Page上建個人部落格，所以也想自己弄一個可以客製化的部落格。

Git & Github

Git是用來管理程式碼的工具，它最主要的特性就是分散式管理(分享程式碼)。而Github是提供Git的一個網路平台，我們可以將自己電腦上的程式碼透過Git建立版本後，發佈到Github上。
