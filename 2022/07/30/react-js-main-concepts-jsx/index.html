概念

顧名思義，JSX 就是 JavaScript XML 的縮寫，是一種 ，讓開發的人可以直接把 HTML 的標籤寫在 JavaScript 裡面。這樣一來，可以省去操作原生 DOM API 的麻煩，讓產生 HTML 的方式變簡單。這種語法被稱作是語法糖(Syntax Sugar)。

JSX 與樣板語言的比較

和其他樣板語言(例如：Handlebars, Mustache) 不一樣的地方，就是可以直接在其中撰寫 JavaScript 程式碼。

樣板語言一般會要求使用到它們自身的語法，才能顯示變數或是執行運算式；而且，通常是分開撰寫，由 JavaScript 定義規範好的資料結構並傳入樣板中，再以樣板語法按照規範顯示。以下是 Handlebars 的範例：
